## Share AI/BI assets with Databricks One

:::tip
* Create a group dedicated to Databricks One, here are examples for a development workspace:
    * dev-consumers.
        * General use.
    * dev-project_1-consumers.
        * Scoped to a specific project data and AI/BI assets.
    * dev-bu_1-consumers.
        * Scoped to a specific business unit data and AI/BI assets.
* Add users to the group.
* Assign the entitlement of ***data consumer*** to the group as shown in the video.
* In order to use the Dashboards or Genie spaces, the group will also need permissions on:
    * **Data:** select permission on the gold schema.
    * **Dashboards and Genie:** can view or can run permissions.
:::
