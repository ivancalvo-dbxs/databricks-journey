# Motivation

:::info Goal
The goal of this guide is to:

* Show how to effectively combine the Databricks products offering.
* List the best practices for each individual product and setup. 
* List different scenarios and potential solutions that organizations face while creating data+AI solutions.
:::

:::tip We're glad that you choose Databricks
* Data teams should be focus on building solutions, not tools.

This guide will focus on the *how* and *why* on Databricks.

For each Databricks product, the *what is* can be found in the official documentation articles.
:::
