# 9. Data Warehousing

## The gold data in UC is ready to be served

<img src={useBaseUrl('/img/dbsql-warehouse.jpg')} alt="description" width="800"/>

:::success Tips
* Querying data translates to SQL Warehouse.
* Serving data translates to SQL Warehouse.
* Accessing data from third-party or external apps / systems translates to SQL Warehouse.
* Databricks Alerts runs on SQL Warehouse.
* AI/BI (Dashboards and Genie) runs on SQL Warehouse.
:::

:::danger Avoid
* Serving data with a cluster. It's inefficient.
* A cluster should only be used to process data.
:::
